**YOU CAN** include it on your mods / 3d models, whether it freeware / payware.  
but, **YOU CAN'T** sell the model or texture generated as it is.

## Features

- Different Plate types
- Every Region plate
- Highly Customizable
- Normal Map
- Include 3D Models
