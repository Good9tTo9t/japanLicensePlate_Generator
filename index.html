Command

[] = Needed Arguments
() = Optional Arguments
prefix : g!

g!generate [vehType] [plateType] [region] [engine] [hiragana] [serial] (scale) (normal)


main command, generate the plate texture.
vehType = car | bike
plateType = regular | commercial | kregular | kcommercial
region = wards name. use g!region for more info
engine = 3 digit engine classification
hiragana = 1 hiragana character. use g!hiragana for more info
serial = 5 digit - 6 digit serial. accepted character = [0-9] [.] [-] [space]
scale = scale the texture. resolution (default: 660px x 330px) * scale (default: 1).
normal = yes | no. generate normal map.
